# Weekly Fitness Planner

This is a simple web app that helps users plan their weekly workout schedule on a visual calendar.

I made this project to learn the basics of HTML, CSS, and JavaScript, while also creating something actually usable (at least for me) and scalable instead of a demo.

---

## What it is

- Displays a customizable weekly calendar (either empty by default or your previously auto-saved workoutplan)
- Shows all available workouts grouped by category (legs, arms, cardio, etc.)
- Lets you drag workouts into days of the week
- Allows you to create and save custom workouts
- Saves your plan automatically using browser localStorage
- Can auto-generate a weekly plan based on a specific fitness goal

---

## Tech used

- HTML
- CSS
- Vanilla JavaScript
- Browser Drag & Drop API
- localStorage

(No frameworks)

---

## Current features

- Drag-and-drop workout scheduling  
- User can create custom workouts
- Weekly plan persistence  
- Goal-based plan generation (rule-based)
